Mac Android Studio完全卸载指南

如果想重新安装Android Studio，记得备份Studio的设置

导出：File/Exporte Settings...
导入：File/Import Settings...

首页的config里也有这两个选项

1.卸载 Android Studio

rm -Rf /Applications/Android\ Studio.app  
rm -Rf ~/Library/Preferences/AndroidStudio*  
rm ~/Library/Preferences/com.google.android.studio.plist  
rm -Rf ~/Library/Application\ Support/AndroidStudio*  
rm -Rf ~/Library/Logs/AndroidStudio*  
rm -Rf ~/Library/Caches/AndroidStudio*

2.删除 gradle

rm -Rf ~/.gradle

3.删除 Android Virtual Devices(AVDs) and *.keystore

rm -Rf ~/.android

4.删除 Android SDK Tools

rm -Rf ~/Library/Android*

5.删除 Projects

rm -Rf ~/yourworkspace/AndroidStudioProjects
